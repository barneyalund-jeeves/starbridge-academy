# Starbridge Academy

A self-contained 5th-grade learning game for Matthew.

## Play Locally

```bash
python3 -m http.server 8090
```

Open `http://localhost:8090` from this folder.

## Features

- Canvas exploration game with keyboard and touch controls
- 5th-grade math, science, vocabulary, history/geography, and Mandarin questions
- Six sectors, puzzle gates, jade collectibles, drones, relics, badges, and hidden Easter eggs
- Generated background art in `assets/starbridge-bg.png`
- WebAudio music and sound effects after first interaction
- Local progress, parent dashboard, CSV export, calm motion mode
